# Astrology Website

## Description
This is an Astrology web application developed to provide users with basic astrology information. The application allows users to view horoscope details, zodiac sign information, and astrological predictions through a simple and user-friendly interface.

## Features
- User-friendly interface
- Zodiac sign information
- Daily horoscope
- Astrology predictions
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
